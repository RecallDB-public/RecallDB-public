# RecallDB Sources

RecallDB v1 uses official U.S. federal recall publications only.

| Source | URL | License / status | Contributes |
| :--- | :--- | :--- | :--- |
| CPSC SaferProducts Recall API | https://www.saferproducts.gov/RestWebServices/Recall | U.S. federal public domain | Consumer product recalls, hazards, remedies, product names |
| NHTSA recall flat files | https://static.nhtsa.gov/odi/ffdd/rcl/ | U.S. federal public domain | Vehicle and equipment recall campaigns |
| openFDA enforcement APIs | https://api.fda.gov/ | U.S. federal public domain | Food, drug and device enforcement recalls |
| FDA Safety Alerts XLSX | https://www.fda.gov/safety/recalls-market-withdrawals-safety-alerts/datatables-data?_format=xlsx | U.S. federal public domain | Cosmetics, animal/veterinary, biologics and other alert enrichment |
| USDA FSIS Recall API | https://www.fsis.usda.gov/fsis/api/recall/v/1 | U.S. federal public domain | Meat, poultry and egg product recalls |
| USCG boating recalls | https://www.uscgboating.org/content/recalls.php | U.S. federal public domain | Recreational boating recalls |

No commercial aggregators are used. EPA is deferred until a stable official recall-specific structured source is available.
